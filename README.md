Система управления проектами и разметки изображений 

**технологии**
- Backend: FastAPI (Python 3.10), SQLAlchemy ORM.
- Database: SQLite.
- Frontend: Vanilla JavaScript, HTML5, CSS3.
- DevOps: Docker, Docker Compose 

**запустить проект**
- для запуска необходим установленный Docker и Docker Compose.

**команда в корневой директории:**
- `docker-compose up --build`
- после запуска интерфейс будет доступен по адресу: 
- `http://localhost:8000`

**особенности**
- проекты: 
  - создание именованных папок, удаление проекта вместе со всеми вложенными данными, каскадное удаление в db и удаление файлов с диска.

- загрузка: 
  - поддержка drag/drop и пакетной загрузки через API.

- аннотирование:
  - рисование рамок мышью.

- хранение координат в нормализованном виде.
- удаление рамок кликом по области.
- валидация минимального размера области.
